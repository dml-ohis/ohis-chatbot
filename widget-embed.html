This is an example page showing how to embed the PM Chatbot widget. Look at the bottom-right corner — there is a floating chat button.

How to embed

Add this single script tag to any HTML page, just before the closing </body> tag:

<link rel="stylesheet" href="dist-widget/week-2-project.css">
<script src="dist-widget/pm-chatbot-widget.js"></script>

Build the widget first: npx vite build --mode widget
